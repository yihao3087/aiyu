# 爱语 APP 测试指引

## 静态检查

```bash
flutter pub get
flutter analyze
```

> 若终端提示找不到 `flutter` 命令，请先将 Flutter SDK 加入 `PATH`。

## 自动化测试

```bash
flutter test
```

## 重点手动回归

1. **设备自动登录**：首次启动需自动注册，重启后应直接进入会话列表。  
2. **头像昵称引导**：头像可从相册选择，昵称需要至少包含一个中文字符。  
3. **聊天输入区**：表情、语音切换流畅；长按消息显示悬浮菜单且不会跳出会话。  
4. **语音消息**：录制声波动画显示，发送后可正常播放。  
5. **位置消息**：分享后应出现百度静态图预览，点击进入地图页面。  
6. **语音通话**：悬浮窗/最小化恢复正常，后台切换不会导致来电断线。  
7. **大文件**：发送超过 200MB 的文件需提示体积超限。  
8. **主题与文案**：检查通知栏、提示信息等是否统一为 “爱语”。
